# MOSAIC 2026 Case Competition

This repository hosts the publishable web appendix for the Stanley 1913 finance case work.

## Live appendix

Open `index.html` locally, or use GitHub Pages once the repository is published.

## Included files

- `index.html`: the online finance appendix
- `04_finance_and_modeling/05_stanley_financial_model_v01.xlsx`: finance model workbook
- `04_finance_and_modeling/07_financial_model_output_snapshot_20260407_v01.csv`: output snapshot used by the appendix
- `06_brand_assets/`: Stanley logo assets used by the page
- `07_team_assets/`: normalized team photos used by the page

## Notes

- Internal research notes, Word files, and original case materials are intentionally not included in this publish repo.
- Some reference cards in the appendix are display-only because their original files are private or not suitable for public reposting.
